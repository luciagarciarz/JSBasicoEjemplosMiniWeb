<h2>OPERADORES ARITMÉTICOS</h2>
<br>
<br>
Los operadores aritméticos se usan para realizar operaciones matemáticas básicas entre valores numéricos.
<br>
<br>
Pueden ser:
<ul>
    <li>Suma</li>
    <li>Resta</li>
    <li>Multiplicación</li>
    <li>División</li>
    <li>Módulo o Resto</li>
    <li>Exponenciación</li>
    <li>Incremento</li>
    <li>Decremento</li>
</ul>
